## 讨论会主题
**讲座加讨论会形式，需要大家积极参与，目的是学到东西。**
1. 每周请一位认知计算神经业内人士报告其相关工作，或者
2. 带着大家读一篇脑科学领域内深度学习和模型相关的文章
3. 每次讨论会一个半小时

**来报告的朋友不一定要报告最新的文献，可以读读经典文献；
不一定是研究，也可以是综述和观点类的文章；
也可以讲讲自己的工作；或者介绍一个Tutorial；
或者自己要做一个正式报告（比如组会上或会议上），可以先来这里练个手。**

**讨论会风格**
1. 需要大家积极参与提问和讨论
2. 不是讲座，不必客气，有问题就问
3. 保证有MIC可以讲话，可能的话尽量开摄像头
4. 学术面前人人平等，不要在乎长幼尊卑
5. 保证交流是和讨论会学术相关
6. 注意不要表达鄙视性观点，比如关于性别、种族、年龄、地域、学校、学位和国籍等


**注意！！！！**
1. 这个讨论会并不是面向大家讲解文章和知识，报告人的讲解往往会略过很多基础知识和细节，假设听众已经懂或从事相关方向研究，因此最好的办法就是参加前看下文章来了解基础知识和细节
2. 希望大家在问问题时可以问过后，让其他人有机会问题，不要一直
3. 我们没有义务满足所有人的要求，因为我们是一些人自发组织的讨论会，如果我们有做得不到位的地方欢迎来信建议和指正，也可以选择不参加。
